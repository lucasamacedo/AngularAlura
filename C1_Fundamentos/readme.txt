Para executar o programa, executar os seguintes comandos no cmd (Prompt de Comando), em instâncias separadas e simultâneas:
Dentro da pasta api: npm start
Dentro da pasta alurapic2: ng s -o

Com isso, o site abrirá automaticamente em seu navegador. Você pode acessá-lo, também, através do endereço http://localhost:4200
Para visualizar o que foi feito durante o curso, acesse o endereço http://localhost:4200/user/flavio
